# Quick Start Guide

## 🚀 Run the Application

### Step 1: Start Backend
```bash
cd backend
npm install
npm run dev
```

### Step 2: Start Frontend (New Terminal)
```bash
npm install
npm run dev
```

### Step 3: Access the App
- Open: http://localhost:3000
- Backend: http://localhost:5000
