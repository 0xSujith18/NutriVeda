# 🌿 NutriVeda – Ayurvedic Diet Planner  

NutriVeda is a **full-stack web application** for personalized Ayurvedic diet planning and wellness tracking. It combines ancient Ayurvedic wisdom with modern nutrition tracking to help users achieve holistic health.  

---

## 🚀 Tech Stack  

### Frontend  
- React 18 + TypeScript (Vite)  
- Tailwind CSS + Radix UI  
- Framer Motion (animations)  
- Recharts (charts & graphs)  
- React Router DOM (navigation)  
- Axios (API calls)  

### Backend  
- Node.js + Express.js  
- MongoDB + Mongoose ODM  
- JWT Authentication  
- bcrypt (password hashing)  
- CORS enabled  

---

## ✨ Features  

- 🔐 **Authentication** – Secure login & signup (JWT-based)  
- 🍽️ **Diet Planner** – Personalized meal plans based on your dosha type  
- 🔍 **Food Explorer** – Search foods by dosha, season, and health benefits  
- 📊 **Dashboard** – Track water intake, nutrition, and daily progress  
- 📈 **Progress Tracker** – Log calories, mood, weight, and wellness stats  
- 👤 **Profile Management** – Dosha questionnaire & preferences  
- 📱 **Responsive Design** – Optimized for desktop & mobile devices
